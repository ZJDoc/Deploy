
# 前言

内网穿透，也就是将局域网端口映射到公网。常用的内网穿透方法包括

1. 路由器的虚拟服务器设置
2. 花生壳的内网穿透服务
3. `n2n`的`p2p`节点连接
4. `ssh`端口转发
5. `ngrok`（***推荐***）
